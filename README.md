# crud produtos

Objetivo deste projeto é criar uma aplicação laravel, contendo um crud para gerenciar produtos.

## Schema do banco de dados

Tabela produtos

| Campo | Tipo |
| -- | -- |
| nome  | String |
| preco | Float |
| descricao | Text |

## Requisitos

- php 8.4
- laravel 12
- sqlite 
- Tailwind CSS

## como executar

```composer install```

```php artisan migrate```

```npm install```

```npm run dev```


```php artisan serve```
